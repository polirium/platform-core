# Polirium Core

Polirium Core contains the shared foundation packages used by the Polirium ERP platform:

- `base`: authentication, authorization, module management, dashboard, and core commands
- `ui`: Tabler-based UI components, assets, and frontend libraries
- `media`: media management and file-related services
- `settings`: centralized settings infrastructure
- `support`: shared support traits, base classes, and utilities

This repository is intended to be used as the core layer for modular business modules under `platform/modules`.
